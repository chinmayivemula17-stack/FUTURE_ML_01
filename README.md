# Project Overview
This project focuses on forecasting future sales using Machine Learning techniques and historical business data.

The system analyzes past sales trends, performs data preprocessing, and predicts future sales to help businesses make better decisions regarding inventory management, demand planning, and revenue forecasting.

# Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

# Dataset

Dataset Used:
Sample Superstore Dataset

The dataset contains:
- Order Date
- Sales
- Product Details
- Region Information
- Profit
- Quantity

# Features Implemented

 Data Cleaning  
 Time-Based Feature Engineering  
 Sales Trend Analysis  
 Correlation Heatmap  
 Sales Forecasting using Machine Learning  
 Future Sales Prediction  
 Model Evaluation  
 Business Insights Visualization

# Machine Learning Model Used

Linear Regression

The model was trained using:
- Year
- Month
- Day

to predict:
- Sales

# Model Evaluation Metrics

The following evaluation metrics were used:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

These metrics help measure prediction accuracy and model performance.

# Business Insights

- Certain products contribute more revenue than others.

- Predictive analytics improves business decision-making.
